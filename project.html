MODULE DESCRIPTION

The modules of the projects are

  1. Administrator
  2. User/Student

ADMINISTRATOR

The administrator has the capacity to add,delete and edit subjects with respective questions and answer also can control user acess.

Administrator is responsible for following activities

  1. DASHBOARD:Administrator is responsible for maintaining the dashborad.
  2. SUBJECTS:
    1. ADD SUBJECTS: Here the administrator can add a new subject in the site.
    2. DELETE SUBJECTS:Here the adminitrator can delete an existing subjects from the site.
    3. UPDATE SUBJECTS:Here the Administrator can update exsiting subjects in the site.
  3. QUESTION SETS:
    1. ADD QUESTION SETS: Here the Administrator can add new sets of question in the site.
    2. DELETE QUESTION SETS: Here the administrator can delete an exsiting set of questions from the site.
    3. UPDATE QUESTION SETS: Here the Administrator can update existing question set information in the site.
  4. CAMPUS:
    1. ADD MOCK TESTS: Here the Administrator can add a new content for mock test in the site.
    2. DELETE MOCK TESTS: Here the Administrator can delete an existing content from the site.
    3. UPDATE MOCK TESTS: Here the Administrator can update existing mock test information in the site.
  5. USERS:
    1. VIEW USER: Here Administrator can view existing users.
    2. ADD USER: Here the Administrator can add new users.

USER:

  1. REGISTER:In this the user can register himself through sign up option.
  2. VIEW SUBJECTS: In this the user is able to view the desired subjects and test himself through the given question sets.
  3. CAMPUS: In this the user can view the recent placement patterns of some companies also can attempt mock test of various subjects.
  4. BYTE POINT: In this module the user can explore some helpful news feeds and resources on soft skills as well as technical skills.
  5. MY RESULTS: Here the user can check the wise results with respect to time.
